# Solve JPA Repository Issue

## Problem:

Spring Boot fails to start because ProjectRepository bean cannot be found.

## Error:

Parameter 0 of constructor in ProjectService required a bean of type 'ProjectRepository' that could not be found.

## Question:

Why is Spring unable to create the repository bean?

## Solution:

Temporary disable components that depend on JPA repository. Temporarily replace database logic with mock data.
