# Goal

create contact message entity.

## Fields:

- id
- name
- email
- message
- createAt

## DTO salidates:

- email
- notBlank

## Repository

- JpaRepository

## Service layer:

- dto->entity->Repository.save()

## Controller:

- POST api/contact
- accept JSON body
- return success message, HTTP 200/201

## Frontend:

- collect form values
- send fetch POST request
- display success/error message

## Supabase:

- create postgreSQL for contact table corresponding to contact entity

## Constrains:

- Keep schema simple
- No authentication
- No relationships
- Do not expose entity directly to frontend input
- No custom queries needed
- Keep mapping manual initially
- Do not return full DB entity unnecessarily
- Keep frontend logic simple
